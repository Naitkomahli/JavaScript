Variabel adalah tempat untuk menyimpan data. Data tersebut dapat berupa angka, teks, atau tipe data lainnya. Variabel memungkinkan kita untuk menyimpan dan mengelola data dengan mudah.

Cara Mendeklarasikan Variabel
Di JavaScript, ada tiga cara untuk mendeklarasikan variabel:
  • var (cara lama, tidak disarankan untuk digunakan lagi).
  • let (digunakan untuk variabel yang nilainya bisa berubah).
  • const (digunakan untuk variabel yang nilainya tetap/tidak bisa diubah).
