Event Handling

Jenis-Jenis Event

Berikut adalah beberapa jenis event yang umum digunakan:

Event Deskripsi
click Saat elemen diklik.
mouseover Saat mouse berada di atas elemen.
mouseout Saat mouse meninggalkan elemen.
input Saat nilai input berubah (untuk <input>).
change Saat nilai input berubah dan kehilangan fokus.
keydown Saat tombol keyboard ditekan.
keyup Saat tombol keyboard dilepas.
load Saat halaman atau elemen selesai dimuat.
submit Saat form di-submit.
