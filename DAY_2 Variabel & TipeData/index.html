Variabel adalah tempat untuk menyimpan data. Data tersebut dapat berupa angka, teks, atau tipe data lainnya. Variabel memungkinkan kita untuk menyimpan dan mengelola data dengan mudah.

Cara Mendeklarasikan Variabel
Di JavaScript, ada tiga cara untuk mendeklarasikan variabel:
  • var (cara lama, tidak disarankan untuk digunakan lagi).
  • let (digunakan untuk variabel yang nilainya bisa berubah).
  • const (digunakan untuk variabel yang nilainya tetap/tidak bisa diubah).

Pertanyaan Umum
Perbedaan let, const, dan var?
  • let : variabel yang bisa diubah nilainya
  • const : Variabel yang tidak bisa diubah nilainya (konstan)
  • var : Cara lama, memiliki scope yang tidak terbatas pada pada block (tidak disarankan)

Kapan menggunakan const
  • Gunakan const ketika nilai variabel tidak akan berubah selama program berjalan

Apa itu scope?
  • Scope adalah area di mana variabel dapat diakses. Variabel let dan const memiliki block scope, sedangkan var memiliki function scope.
