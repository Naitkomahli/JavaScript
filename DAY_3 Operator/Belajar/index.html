Operator Aritmatika

Operator Deskripsi Contoh
+ Penjumlahan 10 + 5 = 15
- Pengurangan 10 - 5 = 5
* Perkalian 10 * 5 = 50
/ Pembagian 10 / 5 = 2
% Modulus 10 % 3 = 1
** Pangkat 2 ** 3 = 8

Operator Perbandingan

Operator Deskripsi Contoh
== Sama dengan 10 == 5 = false
!= Tidak sama dengan 10 != 5 = true
> Lebih besar dari 10 > 5 = true
< Lebih kecil dari 10 < 5 = false
>= Lebih besar sama dengan 10 >= 5 = true
<= Lebih kecil sama dengan 10 <= 5 = false

Operator Logika

Operator Deskripsi Contoh
&& AND(dan) true && false = false
|| OR (atau) true || false→true
! NOT (tidak/bukan) !true → false

Operator Penugasan

Operator Deskripsi Contoh
= Penugasan x = 10
+= Penugasan Penjumlahan x += 5
-= Penugasan Pengurangan x -= 5
*= Penugasan Perkalian x *= 5
/= Penugasan Pembagian x /= 5
%= Penugasan Modulus x %= 5
**= Penugasan Pangkat x **= 5

Operator Lainnya

  • Operator typeof : Digunakan untuk mengecek tipe data.
